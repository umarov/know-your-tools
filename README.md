2. Breadcrumbs
3. Snippets
4. Emmet
5. Smart selection expansion

9. Indenting with Control + [ or ]

    - Being able to use your own shell

    - Command + K Z or Control + K Z

    - Share code
    - Share terminal

15. JS Type checking

16. GitLens

17. Vetur

18. Settings Sync

19. Project Manager

20. Debugging Node and JS projects

22. Auto imports on TypeScript projects
